Issue

Configuration: Laptop (see Configuration section)

After completing a Benchmark test, I dragged the Task Manager window over top of DNSRU. The tabs did not appear to redraw themselves, as seen in the examples below.

and

To repeat the issue:

  1. On the Benchmark tab, click on Test Performance button.
  2. Allow the test to complete on its own.
    Additional info:
        Currently, my BIND-PE install is not answering DNS queries (hence the zero time for execution). This could perhaps be replicated by stopping the service, but I am unable to verify at this time. Note that it is the only server being tested above (127.0.0.1).
  3. Drag Windows Task Manager across window.
